Admin Post Control (Manual Sync)

This application now uses **Manual Sync** via the central **posts.json** file to ensure all devices show the same posts.

Any posts added here would only be visible on *this device* and will not sync.

To add, edit, or delete a post:

  1. Manually edit the **posts.json** file on your computer.
  2. **Redeploy** the entire website to your host.
